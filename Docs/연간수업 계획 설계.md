## 1. 프로젝트 개요

### 1.1 목적
한국 중등학교 교사를 위한 **연간수업계획 자동 생성 및 관리** 시스템 구축

### 1.2 주요 기능
- 학사일정 기반 주차별 수업 가능 시수 자동 계산
- 교육과정 단원 구조 입력 및 주차별 자동 배치
- 계획의 사용자 수정 지원 (자동 + 수동)
- 학기 중 계획 재조정 (학사일정 변경 대응)
- 실제 진행 상황 추적 (LessonLog 연계)

### 1.3 핵심 아이디어
```
Course (수업 정보) + ClassTimetable (학급 시간표)
    ↓
SchoolSchedule (학사일정)
    ↓
주차별 수업 가능 시수 자동 계산
    ↓
CurriculumUnits (단원 구조) 입력
    ↓
주차별 단원 자동 배치
    ↓
사용자 수정 가능 (시수, 단원 배치)
    ↓
LessonLog (실제 수업 기록)으로 진행 상황 추적
```

## 2. 기존 시스템 분석

### 2.1 이미 구현된 데이터베이스 구조

#### Course (수업/강좌)
```sql
CREATE TABLE Course (
    No INTEGER PRIMARY KEY AUTOINCREMENT,
    SchoolCode TEXT NOT NULL,         -- 학교 코드 (FK: School.SchoolCode)
    TeacherID TEXT NOT NULL,          -- 담당 교사 ID (FK: Teacher.TeacherID)
    Year INTEGER NOT NULL,            -- 학년도
    Semester INTEGER NOT NULL,        -- 학기 (1 or 2)
    Grade INTEGER NOT NULL,           -- 대상 학년 (1, 2, 3)
    Subject TEXT NOT NULL,            -- 과목명 (예: "국어", "수학")
    Unit INTEGER NOT NULL,            -- 주당 시수 ⭐
    Type TEXT DEFAULT 'Class',        -- 수업 유형 (Class/Selective/Club)
    Rooms TEXT,                       -- 강의실 목록 (쉼표 구분)
    Remark TEXT                       -- 비고
);
```

**특징:**
- `Unit`: 주당 시수 (핵심 필드)
- `Type`: Class(학급 공통), Selective(선택), Club(동아리)
- `TeacherID`로 교사별 조회 가능
- `Grade`로 학년 단위 관리 (학급 단위 아님)

#### ClassTimetable (학급 시간표)
```sql
CREATE TABLE ClassTimetable (
    No INTEGER PRIMARY KEY AUTOINCREMENT,
    SchoolCode TEXT NOT NULL,
    Year INTEGER NOT NULL,
    Semester INTEGER NOT NULL,
    Grade INTEGER NOT NULL,           -- 학년
    Class INTEGER NOT NULL,           -- 반
    DayOfWeek INTEGER NOT NULL,       -- 요일 (1=월, 2=화, 3=수, 4=목, 5=금)
    Period INTEGER NOT NULL,          -- 교시 (1~7)
    SubjectName TEXT NOT NULL,        -- 과목명
    TeacherName TEXT NOT NULL,        -- 교사명
    Room TEXT                         -- 강의실
);
```

**특징:**
- 학급 관점의 시간표 (학생용)
- `TeacherName`으로 교사 시간표 역추적 가능
- `DayOfWeek`는 정수 (1~5)

#### SchoolSchedule (학사일정 - NEIS 연동)
```sql
CREATE TABLE SchoolSchedule (
    No INTEGER PRIMARY KEY AUTOINCREMENT,
    ATPT_OFCDC_SC_CODE TEXT,          -- 시도교육청코드
    ATPT_OFCDC_SC_NM TEXT,            -- 시도교육청명
    SD_SCHUL_CODE TEXT,               -- 표준학교코드
    SCHUL_NM TEXT,                    -- 학교명
    AY INTEGER,                       -- 학년도
    AA_YMD TEXT,                      -- 학사일자 (yyyyMMdd 형식)
    EVENT_NM TEXT,                    -- 행사명
    EVENT_CNTNT TEXT,                 -- 행사내용
    ONE_GRADE_EVENT_YN INTEGER,       -- 1학년 대상 여부
    TW_GRADE_EVENT_YN INTEGER,        -- 2학년 대상 여부
    THREE_GRADE_EVENT_YN INTEGER,     -- 3학년 대상 여부
    FR_GRADE_EVENT_YN INTEGER,        -- 4학년 대상 여부
    FIV_GRADE_EVENT_YN INTEGER,       -- 5학년 대상 여부
    SIX_GRADE_EVENT_YN INTEGER,       -- 6학년 대상 여부
    SBTR_DD_SC_NM TEXT,               -- 수업공제일명 (해당없음/휴업일/공휴일) ⭐
    IsManual INTEGER DEFAULT 0,       -- 수동 입력 여부
    CreatedAt TEXT,
    UpdatedAt TEXT,
    IsDeleted INTEGER DEFAULT 0       -- Soft Delete
);
```

**특징:**
- NEIS API 연동으로 데이터 자동 수집
- `SBTR_DD_SC_NM`으로 수업 가능 여부 판단 ("휴업일", "공휴일" → 수업 불가)
- `AA_YMD`는 `yyyyMMdd` 문자열 형식
- 학년별 행사 대상 여부 필드 있음

#### LessonLog (수업 기록)
```sql
CREATE TABLE LessonLog (
    No INTEGER PRIMARY KEY AUTOINCREMENT,
    Lesson INTEGER,                   -- 시간표 FK (nullable)
    TeacherID TEXT NOT NULL,          -- 교사 ID
    Year INTEGER NOT NULL,            -- 학년도
    Semester INTEGER NOT NULL,        -- 학기
    Date TEXT NOT NULL,               -- 수업 날짜 (yyyy-MM-dd)
    Period INTEGER,                   -- 교시 (1~7)
    Subject TEXT NOT NULL,            -- 과목명
    Room TEXT,                        -- 강의실 (학급 정보 대용)
    Topic TEXT,                       -- 수업 주제 ⭐ (단원명)
    Content TEXT                      -- 수업 내용
);
```

**특징:**
- `TeacherID` + `Subject` + `Room` 조합으로 특정 학급 수업 식별
- `Topic` 필드에 단원명 기록 → 진행 상황 집계에 활용
- 학급 정보는 `Room` 필드로 간접 관리 (예: "2-3", "3-1")

### 2.2 데이터 관계도
```
Course (수업 정보)
    ├── TeacherID → Teacher
    ├── SchoolCode → School
    └── Unit (주당 시수)
         ↓
ClassTimetable (학급 시간표)
    ├── TeacherName으로 Course와 연계
    └── DayOfWeek + Period → 실제 수업 시간
         ↓
SchoolSchedule (학사일정)
    └── SBTR_DD_SC_NM → 수업 가능 여부
         ↓
LessonLog (수업 기록)
    ├── TeacherID + Subject + Room → 특정 수업 식별
    └── Topic → 단원 진행 상황

⭐ 새로 추가할 테이블들
SubjectYearPlan (연간계획) ←── Course.No 참조
    ↓
WeeklyLessonHours (주차별 시수)
    ↓
CurriculumUnits (단원 구조)
    ↓
WeeklyUnitPlan (주차별 단원 배치)
```

### 2.3 중요 포인트

#### 교사 시간표 구성 방법
현재 코드베이스에는 **교사 관점 시간표**가 별도로 없습니다.
`ClassTimetable`에서 `TeacherName`으로 필터링하여 교사 시간표를 구성해야 합니다.

```csharp
// 교사 시간표 조회 쿼리 (새로 추가 필요)
SELECT DISTINCT DayOfWeek, Period, Grade, Class, SubjectName, Room
FROM ClassTimetable
WHERE SchoolCode = @SchoolCode
  AND Year = @Year
  AND Semester = @Semester
  AND TeacherName = @TeacherName
ORDER BY DayOfWeek, Period;
```

#### 수업 가능일 판단
```csharp
// SchoolSchedule.SBTR_DD_SC_NM 활용
public bool IsHoliday => SBTR_DD_SC_NM == "휴업일" || SBTR_DD_SC_NM == "공휴일";

// 수업 가능일 = IsHoliday가 false인 평일
```

#### 학급별 구분
- `Course`는 학년 단위 (`Grade`)
- 학급 구분은 `ClassTimetable`의 `Grade` + `Class` 조합
- `LessonLog`는 `Room` 필드로 학급 구분 (예: "2-3" = 2학년 3반)

## 3. 요구사항 정의

### 3.1 사용자 워크플로우

#### Phase 1: 학년 초 - 연간수업계획 수립
```
Step 1: Course 선택
   - 교사가 담당하는 Course 목록 조회 (TeacherID 기준)
   - 계획 작성할 Course 선택
   - Course.Unit에서 주당 시수 확인

Step 2: 대상 학급 선택 (Class 타입인 경우)
   - ClassTimetable에서 해당 과목을 가르치는 학급 목록 조회
   - 학급 선택 (예: 2학년 3반)
   - 해당 학급의 실제 시간표 확인

Step 3: 주차별 수업 시수 자동 계산 ⭐
   [자동 계산]
   - ClassTimetable에서 해당 학급의 요일별 시간표 추출
   - SchoolSchedule 기반 학기 주차 분할
   - 각 주의 수업 가능일 계산 (IsHoliday 체크)
   - 학년별 행사 대상 여부 고려
   - 주차별 실제 수업 가능 시수 산출

   [사용자 수정] ✨
   - 특정 주 시수 직접 수정 (추가 수업 시간 포함)
   - 방과후 수업, 0교시, 보충수업 등 자유롭게 추가
   - 수정 후 총 시수 자동 재계산
   - Notes 필드에 추가 사유 메모

Step 4: 교육과정 단원 입력
   - 대단원 > 중단원 > 소단원 구조
   - 각 단원별 예상 시수 입력
   - 단원 순서 지정

Step 5: 단원 자동 배치 ⭐
   [자동 배치]
   - Step 3의 주차별 시수 기반 (추가 시수 포함)
   - 단원 순서대로 주차에 배치
   - 단원 시수가 여러 주에 걸칠 수 있음
   - 한 주에 여러 단원 가능

   [사용자 조정]
   - 단원 순서 변경 (드래그앤드롭)
   - 특정 주에 단원 직접 배치
   - 단원 시수 조정
   - 단원 분할/병합

Step 6: 계획 확정
   - 최종 검토
   - 상태: DRAFT → CONFIRMED
```

#### Phase 2: 학기 중 - 계획 재조정
```
[변동 사항 발생]
- 학사일정 변경 (갑작스런 휴강, 행사 추가)
- 진도 지연/초과

[재조정 프로세스]
Step 1: 변경 감지
   - SchoolSchedule 업데이트 감지
   - 현재 주차 확인

Step 2: 영향 분석
   - 변경된 주차부터 시수 재계산
   - 남은 단원 확인
   - 실제 진행 상황 (LessonLog) 고려

Step 3: 재배치 제안
   - 남은 단원 자동 재배치
   - 조정 내역 요약 표시

Step 4: 사용자 확인 및 수정
   - 제안 검토
   - 필요시 수동 조정
   - 재확정

Step 5: 진행 상황 추적
   - LessonLog 기반 실제 시수 집계
   - 계획 대비 진행률 시각화
   - 지연 구간 알림
```

### 3.2 기능 요구사항

#### FR-1: 주차별 시수 계산 및 관리

**자동 계산:**
- Course + ClassTimetable + SchoolSchedule 기반 자동 계산
- AutoCalculatedHours에 저장 (참고용, 변경 불가)

**사용자 수정:** ✨
- PlannedHours를 자유롭게 수정 가능
- 추가 수업, 보충 수업 등 모두 포함
- 시수 증가/감소 모두 가능
- Notes 필드에 사유 입력 (선택)
- IsModified 플래그로 수정 여부 추적

**일괄 기능:**
- 여러 주에 시수 일괄 추가
- 균등 배분 또는 비율 배분
- 특정 주부터 적용 가능

#### FR-2: 단원 구조 관리
- 대/중/소단원 트리 구조
- 단원별 예상 시수
- 순서 변경 가능

#### FR-3: 단원 배치
- 주차별 시수에 맞춰 자동 배치 (추가 시수 포함)
- 드래그앤드롭으로 수동 배치
- 배치 방식 추적 (IsAutoGenerated)

#### FR-4: 계획 재조정
- 학사일정 변경 감지
- 현재 진행 상황 고려
- 남은 단원 재배치

#### FR-5: 진행 상황 추적
- LessonLog 기반 실제 시수 집계
- 단원별 완료 상황
- 계획 대비 진행률

#### FR-6: 다중 Course/학급 관리
- 한 교사의 여러 Course
- Course별, 학급별 독립적인 계획
- 전체 진도 현황 대시보드

### 3.3 비기능 요구사항

- **성능**: 주차별 계산은 1초 이내
- **사용성**: 자동 계산 후 즉시 수정 가능
- **확장성**: 새로운 학기 복사 기능
- **안정성**: 트랜잭션 처리로 데이터 일관성 보장

## 4. 데이터베이스 설계

### 4.1 신규 테이블 (4개)

#### 4.1.1 SubjectYearPlan (연간수업계획 기본정보)
```sql
CREATE TABLE IF NOT EXISTS SubjectYearPlan (
    No INTEGER PRIMARY KEY AUTOINCREMENT,

    -- Course 참조 (핵심)
    CourseNo INTEGER NOT NULL,

    -- 대상 학급 (Class 타입인 경우)
    TargetGrade INTEGER NOT NULL,     -- 학년
    TargetClass INTEGER,              -- 반 (nullable, 전체 학년인 경우)

    -- 기본 정보
    Year INTEGER NOT NULL,
    Semester INTEGER NOT NULL,
    WeeklyHours INTEGER NOT NULL,     -- 주당 시수 (Course.Unit 기반)
    TotalPlannedHours INTEGER,        -- 전체 계획 시수 (자동 계산)

    -- 상태 관리
    Status TEXT DEFAULT 'DRAFT',      -- DRAFT, CONFIRMED, ADJUSTED

    -- 메타 정보
    CreatedAt TEXT NOT NULL,
    UpdatedAt TEXT NOT NULL,

    FOREIGN KEY (CourseNo) REFERENCES Course(No) ON DELETE CASCADE,
    UNIQUE(CourseNo, TargetGrade, TargetClass)
);

-- 인덱스
CREATE INDEX IF NOT EXISTS idx_yearplan_course
    ON SubjectYearPlan(CourseNo);
CREATE INDEX IF NOT EXISTS idx_yearplan_status
    ON SubjectYearPlan(Year, Semester, Status);
```

**설명:**
- 한 Course + 학급당 하나의 연간계획
- `TargetClass`가 NULL이면 학년 전체 대상
- Course에서 교사, 과목, 주당 시수 정보 획득
- 상태 관리: 작성중 → 확정 → 조정됨

#### 4.1.2 WeeklyLessonHours (주차별 수업 시수)
```sql
CREATE TABLE IF NOT EXISTS WeeklyLessonHours (
    No INTEGER PRIMARY KEY AUTOINCREMENT,

    -- 계획 참조
    YearPlanNo INTEGER NOT NULL,

    -- 주차 정보
    Week INTEGER NOT NULL,            -- 1, 2, 3, ... 주차
    WeekStartDate TEXT NOT NULL,      -- 주 시작일 (yyyy-MM-dd)
    WeekEndDate TEXT NOT NULL,        -- 주 종료일 (yyyy-MM-dd)

    -- 시수 정보
    AutoCalculatedHours INTEGER NOT NULL, -- 자동 계산된 시수 (참고용)
    PlannedHours INTEGER NOT NULL,    -- 실제 계획 시수 (사용자 수정 가능) ✨
    ActualHours INTEGER DEFAULT 0,    -- 실제 진행 시수 (LessonLog 집계)

    -- 수정 추적
    IsModified INTEGER DEFAULT 0,     -- 사용자가 수정했는지 (0=자동, 1=수동)

    -- 메모
    Notes TEXT,                       -- 추가 수업 사유 등

    FOREIGN KEY (YearPlanNo) REFERENCES SubjectYearPlan(No) ON DELETE CASCADE,
    UNIQUE(YearPlanNo, Week)
);

-- 인덱스
CREATE INDEX IF NOT EXISTS idx_weekly_hours_plan
    ON WeeklyLessonHours(YearPlanNo, Week);
```

**설명:**
- 각 주차별로 수업 가능 시수 기록
- `AutoCalculatedHours`: ClassTimetable + SchoolSchedule 기반 자동 계산 (변하지 않음)
- `PlannedHours`: **사용자가 자유롭게 수정 가능** (추가 수업 포함)
- `ActualHours`: LessonLog에서 집계
- `IsModified`: 재계산 시 사용자 수정 주차는 유지
- `Notes`: "방과후 2시간 추가", "0교시 1시간" 등

#### 4.1.3 CurriculumUnits (교육과정 단원)
```sql
CREATE TABLE IF NOT EXISTS CurriculumUnits (
    No INTEGER PRIMARY KEY AUTOINCREMENT,

    -- 계획 참조
    YearPlanNo INTEGER NOT NULL,

    -- 단원 구조
    UnitOrder INTEGER NOT NULL,       -- 단원 순서 (1, 2, 3, ...)
    LargeUnit TEXT,                   -- 대단원 (예: "I. 수와 식")
    MediumUnit TEXT,                  -- 중단원 (예: "1. 다항식")
    SmallUnit TEXT,                   -- 소단원 (예: "01. 다항식의 덧셈")

    -- 시수 정보
    EstimatedHours INTEGER NOT NULL,  -- 예상 소요 시수
    CompletedHours INTEGER DEFAULT 0, -- 실제 완료 시수 (LessonLog 집계)

    FOREIGN KEY (YearPlanNo) REFERENCES SubjectYearPlan(No) ON DELETE CASCADE
);

-- 인덱스
CREATE INDEX IF NOT EXISTS idx_curriculum_order
    ON CurriculumUnits(YearPlanNo, UnitOrder);
```

**설명:**
- 교과서 단원 구조 입력
- 대/중/소단원 계층 구조
- `UnitOrder`: 단원 순서 (사용자가 변경 가능)
- `EstimatedHours`: 단원에 할당할 시수
- `CompletedHours`: 실제 진행된 시수 (LessonLog 집계)

#### 4.1.4 WeeklyUnitPlan (주차별 단원 배치)
```sql
CREATE TABLE IF NOT EXISTS WeeklyUnitPlan (
    No INTEGER PRIMARY KEY AUTOINCREMENT,

    -- 참조
    YearPlanNo INTEGER NOT NULL,
    Week INTEGER NOT NULL,            -- 주차
    UnitNo INTEGER NOT NULL,          -- 단원 참조

    -- 배치 정보
    AllocatedHours INTEGER NOT NULL,  -- 해당 주에 이 단원에 할당된 시수
    OrderInWeek INTEGER NOT NULL,     -- 같은 주에 여러 단원 있을 경우 순서

    -- 생성 방식 추적
    IsAutoGenerated INTEGER DEFAULT 1,-- 0=수동 배치, 1=자동 배치

    -- 상태
    Status TEXT DEFAULT 'PLANNED',    -- PLANNED, IN_PROGRESS, COMPLETED

    FOREIGN KEY (YearPlanNo) REFERENCES SubjectYearPlan(No) ON DELETE CASCADE,
    FOREIGN KEY (UnitNo) REFERENCES CurriculumUnits(No) ON DELETE CASCADE
);

-- 인덱스
CREATE INDEX IF NOT EXISTS idx_unit_plan_week
    ON WeeklyUnitPlan(YearPlanNo, Week);
CREATE INDEX IF NOT EXISTS idx_unit_plan_unit
    ON WeeklyUnitPlan(YearPlanNo, UnitNo);
```

**설명:**
- 어느 주에 어떤 단원을 몇 시간 진행할지 기록
- 한 단원이 여러 주에 걸칠 수 있음 (예: 5주차 3시간, 6주차 2시간)
- 한 주에 여러 단원 가능 (예: 10주차에 단원A 1시간, 단원B 3시간)
- `IsAutoGenerated`: 재조정 시 자동 배치된 것만 재배치
- `OrderInWeek`: 같은 주 내 단원 진행 순서

### 4.2 기존 테이블 활용

| 테이블 | 용도 | 연계 방법 |
|--------|------|-----------|
| **Course** | 수업 기본 정보 | `CourseNo`로 참조, `Unit`으로 주당 시수 |
| **ClassTimetable** | 실제 시간표 | `TeacherName` + `SubjectName` 조인 |
| **SchoolSchedule** | 학사일정 | `SBTR_DD_SC_NM`으로 수업 가능일 판단 |
| **LessonLog** | 실제 진행 시수 | `TeacherID` + `Subject` + `Room` 집계 |

### 5.1 주차별 시수 계산 로직

#### 5.1.1 WeeklyHoursCalculator
```csharp
using System;
using System.Collections.Generic;
using System.Linq;
using System.Threading.Tasks;
using NewSchool.Models;
using NewSchool.Repositories;

namespace NewSchool.Services
{
    /// <summary>
    /// 주차별 수업 시수 계산기
    /// Course + ClassTimetable + SchoolSchedule 기반 자동 계산
    /// </summary>
    public class WeeklyHoursCalculator
    {
        private readonly ClassTimetableRepository _timetableRepo;
        private readonly SchoolScheduleRepository _scheduleRepo;
        private readonly CourseRepository _courseRepo;

        public WeeklyHoursCalculator(
            ClassTimetableRepository timetableRepo,
            SchoolScheduleRepository scheduleRepo,
            CourseRepository courseRepo)
        {
            _timetableRepo = timetableRepo;
            _scheduleRepo = scheduleRepo;
            _courseRepo = courseRepo;
        }

        /// <summary>
        /// Course + 학급 기반 주차별 수업 시수 자동 계산
        /// </summary>
        public async Task<List<WeeklyLessonHours>> CalculateAsync(
            int courseNo,
            int yearPlanNo,
            int targetGrade,
            int? targetClass,
            DateTime semesterStart,
            DateTime semesterEnd)
        {
            // 1. Course 정보 조회
            var course = await _courseRepo.GetByIdAsync(courseNo);
            if (course == null)
                throw new ArgumentException($"Course를 찾을 수 없습니다: No={courseNo}");

            // 2. 해당 학급의 시간표 조회 (교사 관점)
            var timetables = await GetTeacherTimetableAsync(
                course.SchoolCode,
                course.Year,
                course.Semester,
                course.Subject,
                targetGrade,
                targetClass);

            // 3. 학사일정 조회
            var schedules = await _scheduleRepo.GetBySchoolYearAsync(
                course.SchoolCode,
                course.Year);

            // 4. 학기 주차 분할
            var weeks = GetSemesterWeeks(semesterStart, semesterEnd);

            // 5. 주차별 시수 계산
            var weeklyHours = new List<WeeklyLessonHours>();

            foreach (var week in weeks)
            {
                int calculatedHours = CalculateHoursForWeek(
                    week,
                    timetables,
                    schedules,
                    targetGrade);

                weeklyHours.Add(new WeeklyLessonHours
                {
                    YearPlanNo = yearPlanNo,
                    Week = week.Number,
                    WeekStartDate = week.StartDate.ToString("yyyy-MM-dd"),
                    WeekEndDate = week.EndDate.ToString("yyyy-MM-dd"),
                    AutoCalculatedHours = calculatedHours,
                    PlannedHours = calculatedHours,  // 초기값 (사용자 수정 가능)
                    ActualHours = 0,
                    IsModified = 0
                });
            }

            return weeklyHours;
        }

        /// <summary>
        /// 교사 관점 시간표 조회 (ClassTimetable에서 추출)
        /// </summary>
        private async Task<List<ClassTimetable>> GetTeacherTimetableAsync(
            string schoolCode,
            int year,
            int semester,
            string subject,
            int grade,
            int? classNo)
        {
            // ClassTimetable에서 해당 과목 + 학급의 시간표 조회
            var allTimetables = await _timetableRepo.GetByGradeAsync(
                schoolCode, year, semester, grade);

            // 과목명으로 필터링
            var filtered = allTimetables
                .Where(t => t.SubjectName == subject);

            // 특정 반인 경우 반으로 추가 필터링
            if (classNo.HasValue)
            {
                filtered = filtered.Where(t => t.Class == classNo.Value);
            }

            return filtered.ToList();
        }

        /// <summary>
        /// 특정 주의 수업 가능 시수 계산
        /// </summary>
        private int CalculateHoursForWeek(
            WeekInfo week,
            List<ClassTimetable> timetables,
            List<SchoolSchedule> schedules,
            int targetGrade)
        {
            int hours = 0;

            // 주의 각 날짜를 순회
            for (var date = week.StartDate; date <= week.EndDate; date = date.AddDays(1))
            {
                // 주말 제외
                if (date.DayOfWeek == DayOfWeek.Saturday ||
                    date.DayOfWeek == DayOfWeek.Sunday)
                    continue;

                // 수업 가능일인지 확인
                if (!IsClassDay(date, schedules, targetGrade))
                    continue;

                // 해당 요일의 시간표 시수 추가
                int dayOfWeek = GetDayOfWeekNumber(date.DayOfWeek);
                hours += timetables.Count(t => t.DayOfWeek == dayOfWeek);
            }

            return hours;
        }

        /// <summary>
        /// 특정 날짜가 수업 가능일인지 판단
        /// </summary>
        private bool IsClassDay(DateTime date, List<SchoolSchedule> schedules, int targetGrade)
        {
            // 해당 날짜의 학사일정 찾기
            var daySchedules = schedules.Where(s =>
                s.AA_YMD.Date == date.Date && !s.IsDeleted).ToList();

            // 휴업일/공휴일 체크
            if (daySchedules.Any(s => s.IsHoliday))
                return false;

            // 학년별 행사 체크 (해당 학년만 대상인 행사가 있으면 수업 불가)
            var gradeSpecificEvents = daySchedules.Where(s =>
                IsGradeSpecificEvent(s, targetGrade)).ToList();

            if (gradeSpecificEvents.Any())
            {
                // 현장체험학습 등 학년 대상 행사가 있으면 수업 불가
                return false;
            }

            return true;
        }

        /// <summary>
        /// 특정 학년 대상 행사인지 확인
        /// </summary>
        private bool IsGradeSpecificEvent(SchoolSchedule schedule, int grade)
        {
            // 해당 학년이 대상인 행사인지 확인
            bool isTarget = grade switch
            {
                1 => schedule.ONE_GRADE_EVENT_YN,
                2 => schedule.TW_GRADE_EVENT_YN,
                3 => schedule.THREE_GRADE_EVENT_YN,
                4 => schedule.FR_GRADE_EVENT_YN,
                5 => schedule.FIV_GRADE_EVENT_YN,
                6 => schedule.SIX_GRADE_EVENT_YN,
                _ => false
            };

            // 전체 학년 대상이 아니고, 해당 학년만 대상인 경우
            bool allGrades = schedule.ONE_GRADE_EVENT_YN &&
                            schedule.TW_GRADE_EVENT_YN &&
                            schedule.THREE_GRADE_EVENT_YN;

            return isTarget && !allGrades;
        }

        /// <summary>
        /// 학기를 주 단위로 분할
        /// </summary>
        private List<WeekInfo> GetSemesterWeeks(DateTime start, DateTime end)
        {
            var weeks = new List<WeekInfo>();
            int weekNumber = 1;

            // 첫 주 시작일 (해당 주의 월요일)
            var weekStart = start;
            // 월요일로 정렬
            while (weekStart.DayOfWeek != DayOfWeek.Monday && weekStart > start.AddDays(-7))
            {
                weekStart = weekStart.AddDays(-1);
            }
            if (weekStart < start)
                weekStart = start;

            while (weekStart <= end)
            {
                var weekEnd = weekStart.AddDays(6); // 일요일
                if (weekEnd > end)
                    weekEnd = end;

                weeks.Add(new WeekInfo
                {
                    Number = weekNumber++,
                    StartDate = weekStart,
                    EndDate = weekEnd
                });

                weekStart = weekStart.AddDays(7);
            }

            return weeks;
        }

        /// <summary>
        /// System.DayOfWeek → ClassTimetable.DayOfWeek (1~5)
        /// </summary>
        private int GetDayOfWeekNumber(DayOfWeek dayOfWeek)
        {
            return dayOfWeek switch
            {
                DayOfWeek.Monday => 1,
                DayOfWeek.Tuesday => 2,
                DayOfWeek.Wednesday => 3,
                DayOfWeek.Thursday => 4,
                DayOfWeek.Friday => 5,
                _ => 0
            };
        }

        /// <summary>
        /// 재계산 (학사일정 변경 시)
        /// 사용자가 수정한 주차(IsModified=1)는 유지
        /// </summary>
        public async Task<List<WeeklyLessonHours>> RecalculateAsync(
            int yearPlanNo,
            int fromWeek,
            List<WeeklyLessonHours> existingHours,
            int courseNo,
            int targetGrade,
            int? targetClass,
            DateTime semesterStart,
            DateTime semesterEnd)
        {
            // 새로 계산
            var newHours = await CalculateAsync(
                courseNo, yearPlanNo, targetGrade, targetClass,
                semesterStart, semesterEnd);

            // 사용자가 수정한 주차는 유지
            foreach (var existing in existingHours.Where(e => e.Week >= fromWeek))
            {
                var newWeek = newHours.FirstOrDefault(n => n.Week == existing.Week);
                if (newWeek != null && existing.IsModified == 1)
                {
                    newWeek.PlannedHours = existing.PlannedHours;
                    newWeek.IsModified = 1;
                    newWeek.Notes = existing.Notes;
                }
            }

            return newHours;
        }
    }

    /// <summary>
    /// 주차 정보 헬퍼 클래스
    /// </summary>
    public class WeekInfo
    {
        public int Number { get; set; }
        public DateTime StartDate { get; set; }
        public DateTime EndDate { get; set; }
    }
}
```

### 5.2 단원 배치 로직

#### 5.2.1 UnitPlanGenerator
```csharp
using System;
using System.Collections.Generic;
using System.Linq;
using NewSchool.Models;

namespace NewSchool.Services
{
    /// <summary>
    /// 단원 자동 배치 생성기
    /// </summary>
    public class UnitPlanGenerator
    {
        /// <summary>
        /// 단원 자동 배치 (PlannedHours 기준)
        /// </summary>
        public List<WeeklyUnitPlan> Generate(
            int yearPlanNo,
            List<WeeklyLessonHours> weeklyHours,
            List<CurriculumUnit> units)
        {
            var plan = new List<WeeklyUnitPlan>();
            int currentWeekIndex = 0;

            // 단원을 순서대로 배치
            foreach (var unit in units.OrderBy(u => u.UnitOrder))
            {
                int remainingHours = unit.EstimatedHours;
                int orderInWeek = GetNextOrderInWeek(plan, weeklyHours[currentWeekIndex].Week);

                while (remainingHours > 0 && currentWeekIndex < weeklyHours.Count)
                {
                    var week = weeklyHours[currentWeekIndex];

                    // 해당 주에 사용 가능한 시수 (PlannedHours 사용)
                    int usedHours = plan
                        .Where(p => p.Week == week.Week)
                        .Sum(p => p.AllocatedHours);
                    int availableHours = week.PlannedHours - usedHours;

                    if (availableHours <= 0)
                    {
                        currentWeekIndex++;
                        orderInWeek = 1;
                        continue;
                    }

                    // 이번 주에 할당할 시수
                    int hoursThisWeek = Math.Min(remainingHours, availableHours);

                    plan.Add(new WeeklyUnitPlan
                    {
                        YearPlanNo = yearPlanNo,
                        Week = week.Week,
                        UnitNo = unit.No,
                        AllocatedHours = hoursThisWeek,
                        OrderInWeek = orderInWeek++,
                        IsAutoGenerated = 1,
                        Status = "PLANNED"
                    });

                    remainingHours -= hoursThisWeek;

                    // 해당 주 시수를 다 사용했으면 다음 주로
                    if (hoursThisWeek == availableHours)
                    {
                        currentWeekIndex++;
                        orderInWeek = 1;
                    }
                }
            }

            return plan;
        }

        /// <summary>
        /// 특정 주의 다음 순서 번호
        /// </summary>
        private int GetNextOrderInWeek(List<WeeklyUnitPlan> plan, int week)
        {
            var existing = plan.Where(p => p.Week == week).ToList();
            return existing.Count > 0 ? existing.Max(p => p.OrderInWeek) + 1 : 1;
        }

        /// <summary>
        /// 재배치 (특정 주부터)
        /// 수동 배치(IsAutoGenerated=0)는 유지
        /// </summary>
        public List<WeeklyUnitPlan> Regenerate(
            int yearPlanNo,
            int fromWeek,
            List<WeeklyLessonHours> updatedWeeklyHours,
            List<CurriculumUnit> units,
            List<WeeklyUnitPlan> existingPlan)
        {
            // 이미 완료된 단원 필터링
            var completedUnitNos = units
                .Where(u => u.CompletedHours >= u.EstimatedHours)
                .Select(u => u.No)
                .ToHashSet();

            var remainingUnits = units
                .Where(u => !completedUnitNos.Contains(u.No))
                .ToList();

            // 각 단원의 남은 시수 계산
            foreach (var unit in remainingUnits)
            {
                var completedInPlan = existingPlan
                    .Where(p => p.UnitNo == unit.No && p.Week < fromWeek)
                    .Sum(p => p.AllocatedHours);

                // 임시로 EstimatedHours 조정 (남은 시수)
                unit.EstimatedHours = Math.Max(0, unit.EstimatedHours - completedInPlan);
            }

            // fromWeek부터 남은 단원 재배치
            var weeklyHours = updatedWeeklyHours
                .Where(w => w.Week >= fromWeek)
                .ToList();

            // 수동 배치된 것은 유지
            var manualPlan = existingPlan
                .Where(p => p.Week >= fromWeek && p.IsAutoGenerated == 0)
                .ToList();

            // 자동 재배치
            var newPlan = Generate(yearPlanNo, weeklyHours, remainingUnits);

            // 수동 배치 + 자동 배치 병합
            return manualPlan
                .Concat(newPlan)
                .OrderBy(p => p.Week)
                .ThenBy(p => p.OrderInWeek)
                .ToList();
        }
    }
}
```

### 5.3 실제 진행 상황 집계

#### 5.3.1 ProgressTracker
```csharp
using System;
using System.Collections.Generic;
using System.Linq;
using System.Threading.Tasks;
using NewSchool.Models;
using NewSchool.Repositories;

namespace NewSchool.Services
{
    /// <summary>
    /// 진행 상황 추적기
    /// LessonLog 기반 실제 시수 집계
    /// </summary>
    public class ProgressTracker
    {
        private readonly LessonLogRepository _logRepo;
        private readonly SubjectYearPlanRepository _yearPlanRepo;
        private readonly WeeklyLessonHoursRepository _hoursRepo;
        private readonly CurriculumUnitRepository _unitRepo;
        private readonly CourseRepository _courseRepo;

        public ProgressTracker(
            LessonLogRepository logRepo,
            SubjectYearPlanRepository yearPlanRepo,
            WeeklyLessonHoursRepository hoursRepo,
            CurriculumUnitRepository unitRepo,
            CourseRepository courseRepo)
        {
            _logRepo = logRepo;
            _yearPlanRepo = yearPlanRepo;
            _hoursRepo = hoursRepo;
            _unitRepo = unitRepo;
            _courseRepo = courseRepo;
        }

        /// <summary>
        /// WeeklyLessonHours의 ActualHours 업데이트
        /// </summary>
        public async Task UpdateActualHoursAsync(int yearPlanNo)
        {
            var plan = await _yearPlanRepo.GetByIdAsync(yearPlanNo);
            if (plan == null) return;

            var course = await _courseRepo.GetByIdAsync(plan.CourseNo);
            if (course == null) return;

            var weeklyHours = await _hoursRepo.GetByYearPlanAsync(yearPlanNo);

            // Room 값 결정 (학급 정보)
            string room = GetRoomFromPlan(plan);

            foreach (var week in weeklyHours)
            {
                var startDate = DateTime.Parse(week.WeekStartDate);
                var endDate = DateTime.Parse(week.WeekEndDate);

                // LessonLog에서 해당 주의 실제 수업 시수 집계
                int actualHours = await CountLessonLogsAsync(
                    course.TeacherID,
                    course.Year,
                    course.Semester,
                    course.Subject,
                    room,
                    startDate,
                    endDate);

                week.ActualHours = actualHours;
                await _hoursRepo.UpdateAsync(week);
            }
        }

        /// <summary>
        /// LessonLog 시수 집계
        /// </summary>
        private async Task<int> CountLessonLogsAsync(
            string teacherId,
            int year,
            int semester,
            string subject,
            string room,
            DateTime startDate,
            DateTime endDate)
        {
            var logs = await _logRepo.GetBySubjectAndRoomAsync(
                teacherId, year, semester, subject, room, 1000);

            return logs.Count(l =>
                l.Date >= startDate && l.Date <= endDate);
        }

        /// <summary>
        /// CurriculumUnits의 CompletedHours 업데이트
        /// </summary>
        public async Task UpdateCompletedHoursAsync(int yearPlanNo)
        {
            var plan = await _yearPlanRepo.GetByIdAsync(yearPlanNo);
            if (plan == null) return;

            var course = await _courseRepo.GetByIdAsync(plan.CourseNo);
            if (course == null) return;

            var units = await _unitRepo.GetByYearPlanAsync(yearPlanNo);
            string room = GetRoomFromPlan(plan);

            var logs = await _logRepo.GetBySubjectAndRoomAsync(
                course.TeacherID, course.Year, course.Semester,
                course.Subject, room, 1000);

            foreach (var unit in units)
            {
                // Topic 필드에서 단원명 매칭
                int completedHours = logs.Count(l =>
                    MatchesTopic(l.Topic, unit));

                unit.CompletedHours = completedHours;
                await _unitRepo.UpdateAsync(unit);
            }
        }

        /// <summary>
        /// Topic이 단원과 매칭되는지 확인
        /// </summary>
        private bool MatchesTopic(string topic, CurriculumUnit unit)
        {
            if (string.IsNullOrEmpty(topic))
                return false;

            // 대단원, 중단원, 소단원 중 하나라도 포함되면 매칭
            if (!string.IsNullOrEmpty(unit.LargeUnit) && topic.Contains(unit.LargeUnit))
                return true;
            if (!string.IsNullOrEmpty(unit.MediumUnit) && topic.Contains(unit.MediumUnit))
                return true;
            if (!string.IsNullOrEmpty(unit.SmallUnit) && topic.Contains(unit.SmallUnit))
                return true;

            return false;
        }

        /// <summary>
        /// 진행률 계산
        /// </summary>
        public async Task<ProgressSummary> GetProgressAsync(int yearPlanNo)
        {
            var weeklyHours = await _hoursRepo.GetByYearPlanAsync(yearPlanNo);
            var units = await _unitRepo.GetByYearPlanAsync(yearPlanNo);

            int totalPlanned = weeklyHours.Sum(w => w.PlannedHours);
            int totalActual = weeklyHours.Sum(w => w.ActualHours);

            int completedUnits = units.Count(u => u.CompletedHours >= u.EstimatedHours);
            int totalUnits = units.Count;

            return new ProgressSummary
            {
                TotalPlannedHours = totalPlanned,
                TotalActualHours = totalActual,
                ProgressRate = totalPlanned > 0 ? (double)totalActual / totalPlanned * 100 : 0,
                CompletedUnits = completedUnits,
                TotalUnits = totalUnits,
                UnitProgressRate = totalUnits > 0 ? (double)completedUnits / totalUnits * 100 : 0
            };
        }

        /// <summary>
        /// 계획에서 Room 값 추출
        /// </summary>
        private string GetRoomFromPlan(SubjectYearPlan plan)
        {
            // TargetClass가 있으면 "학년-반" 형식
            if (plan.TargetClass.HasValue)
            {
                return $"{plan.TargetGrade}-{plan.TargetClass}";
            }
            return string.Empty;
        }
    }

    /// <summary>
    /// 진행 상황 요약
    /// </summary>
    public class ProgressSummary
    {
        public int TotalPlannedHours { get; set; }
        public int TotalActualHours { get; set; }
        public double ProgressRate { get; set; }
        public int CompletedUnits { get; set; }
        public int TotalUnits { get; set; }
        public double UnitProgressRate { get; set; }
    }
```

```csharp
using System.Collections.Generic;
using System.Linq;
using System.Threading.Tasks;
using NewSchool.Models;
using NewSchool.Repositories;

namespace NewSchool.Services
{
    /// <summary>
    /// 주차별 시수 수정 헬퍼
    /// </summary>
    public class WeeklyHoursHelper
    {
        private readonly WeeklyLessonHoursRepository _weeklyHoursRepo;
        private readonly SubjectYearPlanRepository _yearPlanRepo;

        public WeeklyHoursHelper(
            WeeklyLessonHoursRepository weeklyHoursRepo,
            SubjectYearPlanRepository yearPlanRepo)
        {
            _weeklyHoursRepo = weeklyHoursRepo;
            _yearPlanRepo = yearPlanRepo;
        }

        /// <summary>
        /// 일괄 시수 추가
        /// </summary>
        public async Task AddHoursEvenlyAsync(
            int yearPlanNo,
            int additionalHours,
            int? fromWeek = null)
        {
            var weeklyHours = await _weeklyHoursRepo.GetByYearPlanAsync(yearPlanNo);

            // 적용 범위
            var targetWeeks = fromWeek.HasValue
                ? weeklyHours.Where(w => w.Week >= fromWeek.Value).ToList()
                : weeklyHours;

            if (targetWeeks.Count == 0) return;

            // 주당 추가 시수
            int hoursPerWeek = additionalHours / targetWeeks.Count;
            int remainder = additionalHours % targetWeeks.Count;

            for (int i = 0; i < targetWeeks.Count; i++)
            {
                var week = targetWeeks[i];
                int addHours = hoursPerWeek + (i < remainder ? 1 : 0);

                week.PlannedHours += addHours;
                week.IsModified = 1;

                if (addHours > 0 && string.IsNullOrEmpty(week.Notes))
                {
                    week.Notes = $"추가 수업 {addHours}시간";
                }

                await _weeklyHoursRepo.UpdateAsync(week);
            }

            // 전체 총 시수 업데이트
            await UpdateTotalPlannedHoursAsync(yearPlanNo);
        }

        /// <summary>
        /// 특정 주 시수 수정
        /// </summary>
        public async Task UpdateWeekHoursAsync(
            int weeklyHoursNo,
            int newPlannedHours,
            string notes = null)
        {
            var week = await _weeklyHoursRepo.GetByIdAsync(weeklyHoursNo);
            if (week == null) return;

            week.PlannedHours = newPlannedHours;
            week.IsModified = 1;

            if (notes != null)
            {
                week.Notes = notes;
            }

            await _weeklyHoursRepo.UpdateAsync(week);

            // 전체 계획 총 시수 업데이트
            await UpdateTotalPlannedHoursAsync(week.YearPlanNo);
        }

        /// <summary>
        /// 총 계획 시수 재계산
        /// </summary>
        private async Task UpdateTotalPlannedHoursAsync(int yearPlanNo)
        {
            var plan = await _yearPlanRepo.GetByIdAsync(yearPlanNo);
            if (plan == null) return;

            var weeklyHours = await _weeklyHoursRepo.GetByYearPlanAsync(yearPlanNo);

            plan.TotalPlannedHours = weeklyHours.Sum(w => w.PlannedHours);
            await _yearPlanRepo.UpdateAsync(plan);
        }
    }
}
```

```csharp
                using var reader = await cmd.ExecuteReaderAsync();
                if (await reader.ReadAsync())

                    return MapPlan(reader);
                }
                return null;
            }
```
