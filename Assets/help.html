도움말

시작하기

NewSchool은 교사를 위한 학급 경영 및 수업 관리 데스크톱 앱입니다. 학생 정보, 누가 기록, 수업 계획, 일정 관리, 학급 일지 등 교사의 업무를 효율적으로 지원합니다.

최초 설정

앱을 처음 실행하면 초기 설정 화면이 나타납니다.

  1. 학교 검색 — NEIS API로 근무 학교를 검색합니다. 학교명 일부만 입력해도 검색되며, 선택하면 시도교육청 코드와 학교 코드가 자동 설정됩니다.
  2. 사용자 정보 — 이름, 연락처, 이메일, 담당 교과를 입력합니다. 사용자 ID는 자동 생성됩니다.
  3. 학년도/학기 — 현재 학년도와 학기를 설정합니다. 담임 학년/반은 선택사항이며 나중에 변경 가능합니다.

화면 구성

상단 네비게이션 바에서 각 메뉴를 선택할 수 있습니다. 메뉴에 마우스를 올리면 하위 메뉴가 펼쳐집니다.

메뉴설명
홈오늘의 일정, 급식, 학사일정 대시보드
달력일정 및 할일 관리, 구글 캘린더 연동
학급학급 일지, 학생 관리, 누가 기록, 학생부, 자리 배정, 게시판, 시간표
수업수업 관리, 연간 계획, 진도, 수업 기록, 동아리
업무업무 자료 관리, 카테고리별 게시판
아카이브자유 분류 자료 보관함
설정학교, 학사일정, 학생, 앱 설정

권장 초기 설정 순서

  1. 학교 설정에서 학교 정보, 학년도/학기, 교시 시간을 확인합니다.
  2. 학생 관리에서 학생 명단을 등록합니다 (엑셀 가져오기 지원).
  3. 학사일정 관리에서 NEIS 학사일정을 가져옵니다.
  4. 수업(교과) 관리에서 담당 교과를 등록하고 수강생을 배정합니다.
  5. 시간표를 설정하고 연간 수업 계획을 작성합니다.

홈

앱을 실행하면 가장 먼저 보이는 대시보드입니다.

  • 오늘의 일정 — 캘린더에 등록된 오늘의 일정과 할일을 표시합니다.
  • 급식 정보 — NEIS API를 통해 오늘의 급식 메뉴를 자동으로 불러옵니다.
  • 학사일정 — 이번 주/이번 달의 학사일정을 한눈에 확인합니다.

달력

일정과 할일을 관리하는 캘린더입니다. 구글 캘린더 양방향 동기화를 지원합니다.

일정 등록

  • 날짜를 더블클릭하면 새 일정 등록 대화상자가 열립니다.
  • 일정 유형: 할일(체크박스) / 일정(시간 지정) 중 선택합니다.
  • 카테고리: 일정을 분류할 카테고리(캘린더)를 선택합니다.
  • 반복 설정: 매일/매주/매월/매년 반복을 지정할 수 있습니다.
  • 종일 일정: 시간을 지정하지 않는 종일 일정으로 설정할 수 있습니다.

일정 수정/삭제

  • 일정을 클릭하면 수정 대화상자가 열립니다.
  • 할일은 체크박스로 완료 상태를 토글합니다.
  • 삭제 버튼으로 일정을 제거합니다.

구글 캘린더 연동

설정에서 구글 계정을 연결하면 양방향 동기화가 가능합니다.

  • 일정 생성/수정/삭제 시 즉시 동기화
  • 자동 동기화 간격 설정 (기본 15분)
  • 수동 동기화 버튼으로 즉시 반영

우측 패널

선택한 날짜의 일정 목록이 표시됩니다. 카테고리별로 필터링할 수 있습니다.

담임 학급의 전반적인 관리 기능을 제공합니다.

명렬표, 비상연락망, 학생카드를 출력합니다.

학급 일지

매일의 학급 활동을 기록하고 관리합니다.

주요 기능

  • 날짜 이동: 날짜 선택기와 좌/우 화살표로 이동합니다.
  • 서식 편집기: 굵게, 글꼴, 색상, 정렬, 표 등 서식을 지원하는 리치 텍스트 에디터를 사용합니다.
  • 자동 저장: 다른 날짜로 이동하면 현재 내용이 자동 저장됩니다.
  • 당일 학생 기록: 해당 날짜에 작성된 학생 누가 기록이 함께 표시됩니다.

학생 우클릭 메뉴

학생 목록에서 학생을 우클릭하면 다음 메뉴가 나타납니다:

  • 관찰 기록 작성
  • 오늘의 기록 보기
  • 학생 정보 보기

학생 정보

학생의 기본 정보와 프로필을 관리합니다.

화면 구성

  • 좌측: 학생 목록 (학년도/학년/반 필터)
  • 우측: 선택한 학생의 상세 정보 카드

기능

  • 사진 등록: 학생 사진을 업로드하여 프로필에 표시합니다.
  • 연락처/보호자 정보: 학생 및 보호자 연락처를 편집합니다.
  • 학생 등록: 개별 등록 또는 엑셀 일괄 등록을 지원합니다.

학생 기록 (누가 기록)

학생별 관찰 기록을 누적 관리하며, 학기말 학생부 작성의 기초 자료가 됩니다.

기록 조회

  • 학생 선택 후 카테고리별 필터링 (교과, 창의적 체험활동, 행동 등)
  • 기간/학기별 필터링

기록 작성

  • 기본 정보: 학년도, 학기, 날짜, 카테고리
  • 교과활동: 교과명 추가 입력 필드
  • 관찰 내용: 자유 기술 + 바이트 카운터 (NEIS 입력 제한 참고)
  • 활동 상세: 활동명, 유형, 세부영역, 성취도
  • 태그: 검색/분류를 위한 태그 지정

기록 편집/삭제

  • 선택 후 대화상자에서 편집하거나 목록에서 인라인 편집
  • 편집 시 자동으로 수정 체크 표시
  • 여러 기록 선택 후 일괄 삭제 가능

학생부 관리

누가 기록을 바탕으로 학생부 특기사항을 작성하고, 학급 전체를 일괄 관리합니다.

영역별 관리

학년도, 학년, 반, 카테고리를 선택하여 조회합니다.

  • 교과활동 — 교과학습발달상황 (과목별)
  • 개인별세특 — 개인별 세부능력 및 특기사항
  • 자율활동 / 진로활동 / 동아리활동 / 봉사활동 — 창의적 체험활동
  • 종합의견 — 행동특성 및 종합의견

각 영역별 NEIS 바이트 제한(1,500~2,100 Byte)이 실시간으로 표시됩니다. 한글 3바이트, 영문/숫자 1바이트 기준입니다.

버튼 구성

  • 저장 — 체크하거나 수정한 항목을 DB에 저장
  • 삭제 — 선택한 항목을 DB에서 완전 삭제
  • 글꼴 크기 — 목록의 텍스트 크기 조절
  • 일괄 입력 — 학급 전체 학생의 특기사항을 한 화면에서 편집
  • 일괄 출력 — 학급 전체 특기사항을 PDF 또는 엑셀로 한 문서에 출력

일괄 입력

좌측 학생 목록에서 방향키로 학생을 전환하며 특기사항을 편집합니다.

  • 누가기록 참조 — 하단 패널을 열면 해당 학생의 누가기록 초안(DraftSummary)이 영역별로 표시됩니다.
  • 초안 자동 생성 — 누가기록의 학생부 초안을 병합하여 특기사항 텍스트를 자동으로 채웁니다. 기존 내용이 있으면 추가/덮어쓰기를 선택할 수 있습니다.
  • 맞춤법 검사 — 외부 맞춤법 검사기(나라맞춤법검사기)를 엽니다.

일괄 출력

필터 조건을 설정하여 원하는 범위만 출력할 수 있습니다.

  • 형식 — PDF 또는 엑셀 선택
  • 영역 — 특정 영역만 필터링 (전체/자율활동/교과활동 등)
  • 상태 — 전체, 작성중만, 마감만
  • 빈 항목 제외 — 내용이 비어있는 항목을 제외

작업 흐름

  1. 평소 꾸준히 누가 기록 작성
  2. 학기말에 일괄 입력 → 초안 자동 생성 실행
  3. 생성된 문구 검토/수정 (바이트 제한 확인)
  4. 일괄 출력으로 전체 확인 후 NEIS에 입력

자리 배정

시각적 드래그앤드롭으로 교실 좌석을 배치합니다.

기본 사용법

  • 드래그앤드롭: 학생 이름을 끌어다 자리에 놓습니다.
  • 클릭 배치: 왼쪽 목록에서 학생을 선택한 뒤 빈 좌석을 클릭하여 배치합니다.
  • 자동 배정: 랜덤 배정 기능으로 빠르게 배치합니다.

배열 설정

  • 줄 수: NumberBox로 2~8줄 설정 (예: "5줄")
  • 짝: 체크박스로 짝 여부 설정 (체크=2명, 해제=1명)
  • 도트 패턴: 현재 배열을 인라인 점 패턴으로 시각화 (예: ●● ●● ●●)

짝 분리/고정

"● 자리 배정" 제목의 불릿 버튼을 클릭하여 짝 설정 다이얼로그를 엽니다.

  • 분리: 특정 학생 쌍이 인접하지 않도록 자동배정 시 제약을 적용합니다.
  • 고정: 특정 학생 쌍이 반드시 인접하도록 자동배정 시 제약을 적용합니다.

좌석 옵션 (우클릭)

  • 미사용 좌석: 좌석을 사용하지 않도록 설정합니다.
  • 지정 좌석: 특정 학생을 해당 좌석에 고정합니다.
  • 미표시 좌석: 학생이 배정되어 있어도 인쇄 시 해당 좌석을 숨깁니다.

인쇄

현재 배치도를 A4 1장 PDF로 출력합니다.

  • 좌석+교탁이 용지 하단 기준으로 배치되며, 상단에 여백이 남습니다.
  • 사진 표시 체크 시 학생 사진이 3:4 비율로 최대 크기 표시됩니다.
  • 줄 사이에 통로 간격이 삽입되어 줄 구분이 명확합니다.
  • 설명 입력란에 메시지를 입력하면 인쇄물 상단에 표시됩니다.

학급 게시판

학급 자료와 공지사항을 관리하는 게시판입니다.

보기 모드

  • 목록: 제목, 작성자, 날짜 순 리스트 표시
  • 그리드: 카드형 격자 표시
  • 메모: 포스트잇 스타일 표시

글쓰기

  • 서식 편집기 (굵게, 글꼴, 색상, 정렬, 표, 이미지 등)
  • 파일 첨부 (이미지, 문서 등)
  • 카테고리/주제 분류

인쇄

  • 게시물 보기 화면에서 인쇄 버튼을 클릭하면 HTML 서식(표, 이미지, 글꼴, 색상)을 그대로 보존하여 인쇄합니다.
  • 미리보기 창이 화면 중앙에 열리며, 시스템 인쇄 대화상자가 자동 표시됩니다.

명렬표 삽입

학급/수업/동아리 학생 명단을 표 형태로 자동 생성하여 삽입합니다.

  • 범위 선택: 학급, 수업(교과), 동아리 중 선택
  • 레이아웃: 세로형(구분별 그룹) / 가로형(구분별 열)
  • 제목/열 머리글 커스터마이징 가능
  • 학생 번호/이름 자동 채움
  • 게시물 제목이 비어 있으면 표 제목으로 자동 설정

학급 시간표

담임 학급의 주간 시간표를 관리합니다.

  • 셀을 클릭하여 교과를 배정합니다 (등록된 교과 목록 또는 직접 입력).
  • 교시별 시간은 학교 설정의 교시 시간 설정에 따라 자동 계산됩니다.

학생 정보 출력

학생 관련 정보를 인쇄 가능한 형태로 출력합니다.

  • 명렬표: 번호, 이름, 성별 기본 명단
  • 비상연락망: 보호자 연락처 포함 명단
  • 학생카드: 사진, 기본 정보, 메모가 포함된 개인 카드

수업 관리

담당 교과의 수업 전반을 관리합니다.

설정 순서

  1. 수업(교과) 관리에서 담당 교과를 등록합니다.
  2. 수강생을 반 단위 또는 개별로 배정합니다.
  3. 단원 구조를 설정합니다.
  4. 연간 수업 계획을 작성합니다.
  5. 수업 후 수업 기록을 남깁니다.

수업 홈

수업 관련 대시보드입니다.

  • 배정된 교과 목록 표시
  • 오늘의 수업 일정
  • 반별 진도 현황 요약

연간 수업 계획

학기 단위의 수업 계획을 체계적으로 수립합니다.

단원 관리

대단원/소단원 구조로 교육과정을 구성하고, 각 단원에 차시를 배정합니다.

주간 배분

  • 자동 배분: 학사일정 기반으로 실제 수업 가능일을 계산하여 자동 배분합니다.
  • 수동 배분: 교사가 직접 주별 계획을 조정합니다.

진도 관리

수업 진도를 날짜별, 반별로 추적합니다.

  • 매트릭스 형태로 진도를 한눈에 파악합니다.
  • 날짜별 단원/차시 기록
  • 여러 반의 진도를 비교
  • 연간 계획 대비 실제 진도 비교

수업 기록

각 수업 활동의 상세 내용을 기록합니다.

  • 교과, 학급, 단원, 주제, 활동 내용 기록
  • 수업 자료, 학생 결과물, 영상 등 파일 첨부
  • 학생 관찰 기록과 연계하여 수업 중 특이사항을 학생별로 남길 수 있습니다.

수업 시간표

교사 본인의 주간 수업 시간표를 표시합니다.

  • 등록된 교과가 자동 반영됩니다.
  • 어느 반에 몇 교시 수업인지 한눈에 확인할 수 있습니다.

동아리 관리

동아리 활동을 관리합니다.

동아리 운영

  • 동아리 생성: 정규/자율 유형 선택
  • 부원 등록: 반별 또는 개별 학생 등록
  • 활동 기록: 날짜, 주제, 활동 내용, 출석 관리

수업(교과) 관리

담당 교과를 등록하고 관리합니다.

  • 교과 등록: 교과명, 학년, 학기, 주당 수업 시수 설정
  • 수강생 배정: 반 단위 일괄 배정 또는 개별 학생 등록
  • 단원 구조: 대단원/소단원/차시 구조 설정

업무 관리

교무 업무 자료를 카테고리별로 관리하는 게시판입니다.

  • 카테고리별 분류
  • 서식 편집기와 파일 첨부 지원 (학급 게시판과 동일)
  • 목록/그리드/메모 보기 모드

아카이브

자유로운 분류 체계로 자료를 보관합니다.

  • 전년도 자료, 연수 자료, 참고 문서, 개인 메모 등
  • 제목, 내용, 카테고리로 검색
  • 카테고리 자유 생성

설정

앱의 전반적인 설정을 관리합니다. 하위 메뉴로 학교 설정, 학사일정 관리, 학생 관리, 앱 설정이 있습니다.

학교 설정

사용자 / 담임반

  • 교사 이름 설정
  • 담임 학년/반 지정 (선택사항)

학교 정보

  • NEIS 학교 검색으로 자동 설정
  • 시도교육청 코드, 학교 코드, 주소 등 자동 입력
  • 수동 수정도 가능

학년도 및 학기

  • 학년도와 학기(1학기/2학기) 설정
  • 학년도는 매년 3월 기준

학사일정 관리

  • NEIS 가져오기: "NEIS 학사일정 가져오기" 버튼으로 일괄 등록
  • 수동 입력: 개별 학사일정 추가
  • 일괄 삭제: 체크박스로 선택 후 삭제
  • 구글 캘린더 등록: 학사일정을 구글 캘린더에 일괄 등록

학생 관리

엑셀 가져오기

NEIS에서 내보낸 엑셀 파일을 그대로 가져올 수 있습니다.

  • 한글 숫자(일, 이, 삼...)가 포함된 학년/반을 자동 인식합니다.
  • 번호, 이름, 성별 등 기본 정보를 자동 매핑합니다.

학생 등록/관리

  • 개별 등록: 학생 정보를 직접 입력
  • 학급 배정: 학년/반 이동 처리
  • 학생 삭제: 기록은 보존하고 재적 정보만 제거

효율적 사용법

학기 초

  1. 학교 설정 확인 (학년도, 학기, 교시 시간)
  2. 학생 명단 등록 (엑셀 가져오기)
  3. NEIS 학사일정 가져오기
  4. 교과 등록 및 수강생 배정
  5. 시간표 설정
  6. 연간 수업 계획 작성

일상 수업

  • 학급 일지에서 당일 학급 활동 기록 + 학생 우클릭으로 누가 기록 작성
  • 수업 기록에서 수업 활동 + 주목할 학생 관찰 기록 연계

학기말

  1. 누적된 누가 기록 확인
  2. 학생부 AI 자동 생성 실행
  3. 생성된 문구 검토 및 수정
  4. NEIS에 입력

새 학년도

학교 설정에서 학년도를 업데이트하고, 새 학생 명단을 등록하면 됩니다. 이전 학년도 데이터는 그대로 보존됩니다.

문제 해결

증상해결 방법
데이터가 표시되지 않음상단의 학년도/학년/반 필터를 확인하세요.
NEIS 기능 오류설정에서 NEIS API 키가 정상 입력되었는지 확인하세요.
앱이 느림앱 설정 > 성능 및 캐시에서 캐시를 활성화하세요.
오류 발생앱 설정 > 고급에서 로그 레벨을 Debug로 설정 후 로그 폴더를 확인하세요.
